# ROCK-PAPER-SCISSORS
This project is a real-time Stone-Paper-Scissors game built using hand gesture recognition with OpenCV and MediaPipe. It detects player gestures through a webcam and interacts with Arduino for output.
Today’s project showcases an interactive Stone-Paper-Scissors game that combines computer vision and embedded systems. Using OpenCV, live video is captured from the webcam, while MediaPipe is used to accurately detect hand landmarks in real time. By analyzing finger positions, the system identifies whether the user is showing rock, paper, or scissors.

Once the gesture is recognized, the result is processed and can be sent to an Arduino for additional outputs such as LEDs, buzzer, or display. The system then compares the user’s gesture with a randomly generated computer move to determine the winner.

This project demonstrates the integration of real-time image processing, gesture recognition, and hardware interaction. It highlights key concepts such as computer vision, human-computer interaction, and IoT-based extensions. The project is engaging, interactive, and serves as a strong foundation for more advanced AI-based gesture recognition systems.
